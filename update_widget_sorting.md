# update_widget_sorting

Updates how reward campaigns, quests, and redemption options are sorted in the widget. Send ONLY the fields you want to change.

## Fields

- `challengeSorting` (string) ‚Äî how campaigns are sorted: `"Creation Date"` or `"Name"`
- `questSorting` (string) ‚Äî how quests are sorted: `"Creation Date"` or `"Name"`
- `redemptionSorting` (string) ‚Äî how redemption options are sorted: `"Creation Date"` or `"Points"`

## Note

`"Manual"` sorting is **not supported via MCP** ‚Äî use the dashboard for drag-drop ordering.
